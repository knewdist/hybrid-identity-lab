# Hybrid Identity Lab

Windows Server 2025 + Active Directory + NTFS + Microsoft 365 Hybrid Identity Deep Dive.

This lab builds:

- Windows Server Domain Controller (DC01)
- File Server (FS01)
- Windows 11 Enterprise Client (CL01)
- On-prem AD DS + DNS
- NTFS & Share permissions model
- Group Policy deployment
- Microsoft 365 tenant integration
- Hybrid identity sync

This is a structured, MSP-style infrastructure lab with break/fix scenarios and documented evidence.
